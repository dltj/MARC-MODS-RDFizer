  What is this?
  ------------

This is an utility tool to convert MARC and/or MODS to RDF 


  How do I use it?
  ----------------
  
This is a command line program, so open a command shell and type:

 ./marcmods2rdf.sh  [from unix]
 .\marcmods2rdf.bat [from windows]

and follow the instructions on the help screen.
